SMS Marketing, Text Marketing and Text Messaging service through TXTImpact plugin allows to collect mobile numbers, name, and emails from blog or website and send text messages to the subscribers.


== Description ==

<a href="https://www.txtimpact.com/sms-marketing/">SMS Marketing </a> and <a href="https://www.txtimpact.com/">Text Marketing </a> plugin through TXTImpact allows blog's visitors to sign up to receive text message notifications whenever a new post is added. The plug-in integrates with TXTImpact Text Marketing and Text Messaging platform and allows users to signup to mobile list via text message using a keyword on a short code or local phone number. You can send text messages to opt-in mobile subscriber from wordpress admin panel as required.

<b>More Information</b>

Add text messaging to your application using TXTImpact <a href="https://www.txtimpact.com/sms-gateway-api/">SMS API</a>.

Visit the <a href="https://www.txtimpact.com/">Text Marketing</a> for more information.

TXTImpact also offers <a href="https://www.txtimpact.com/email2sms/">Email To Text Messaging Service </a> allows business to send text message from outlook, and any email client.


== Installation ==

1. Upload **txtimpact-sms folder** to the ** /wp-content/plugins/ ** directory
1. Activate the plugin through the 'Plugins' menu in WordPress

= Add widget to sidebar =

    1. Go to Apearance -> Widgets in your WordPress dashboard
    1. Drag the SMS Subscribe widget to your desired widget area


== Frequently Asked Questions ==

= How do I get started? =

You will need a TXTImpact.com account to be able to send automatic or manual sms alerts. Visit https://www.txtimpact.com/wordpress-plugin/ for more information.

= How much does TXTImpact.com service cost? =

You can find TXTImpact.com SMS gateway and text message marketing pricing information at https://www.txtimpact.com/pricing-page/

= What countries is TXTImpact.com supports? =

TXTImpact provide service The United States, Canada, UK and to many other countries. For Global coverage, visit https://www.txtimpact.com/international/

= How to enable new post sms notification? =

Go to TXTImpact.com SMS tab and click on Settings page. Check the "Notify SMS subscribers when a new post is published" option.

= How to send manually a SMS to all subscribers? =

To send sms or text message to all subscribers, go to TXTImpact SMS tab and click "Send SMS". On "Send SMS Message" page you can compose and send a text message up to 160 characters.

= Can subscribers Reply to the SMS or text message? =

Yes, subscriber can reply to the text messages, you can see all received text messages on received messages page.

= Can someone subscribe to the mobile list by sending text message? =

Yes, the plug-in is integrated with TXTImpact <a href="https://www.txtimpact.com/sms-marketing/">sms marketing platform </a> to receive text messages from users. You can setup a keyword on TXTImpact platform and configure your blog sms Callback URL to allow users to subscribe to your mobile list.
